Multiple Page Applications (MPA) load a new HTML page from the server whenever the user navigates to a different page, which causes the entire page to reload.

/*Single Page Applications*/ (SPA) load only one HTML page initially, and when the user navigates within the application, JavaScript dynamically updates the content without reloading the entire page. This makes SPA faster and provides a smoother user experience.

**Reconciliation** is the process by which React compares the previous Virtual DOM with the new Virtual DOM and updates only the changed parts in the real DOM.
**Virtual DOM** is an in-memory representation of the real DOM that React uses to optimize updates and improve performance.
What is Hooks?
Hooks are special function provided by react that allows function components to use react features like state,lifecycle methods and other react features without writing a class component.
